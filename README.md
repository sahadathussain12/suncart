# 🌞 SunCart – Summer Essentials Store

A modern summer eCommerce website built with Next.js where users can browse summer products, view details, and authenticate securely.

## 🔗 Live Link

https://suncart-kuxb.vercel.app/

## 💻 GitHub Repository

https://github.com/sahadathussain12/suncart

## 🎯 Purpose

SunCart helps users explore and purchase summer essentials like sunglasses, skincare, beach accessories, and more.

## ✨ Key Features

- Responsive design
- User Authentication (BetterAuth)
- Google Sign In
- Protected Product Details Page
- User Profile & Update Profile
- Summer Products Showcase

## 🛠️ Technologies Used

- Next.js
- Tailwind CSS
- HeroUI
- BetterAuth

## 📦 NPM Packages

- @heroui/react
- better-auth
- react-icons
- react-toastify
